# reflow_oven

Partlist:
- Arduino Nano
- HLK-PM01 5V power supply
- MAX31855 module with K type thermocouple
- FOTEK SSR-40DA
- rotary encoder
- 2.4" SPI TFT display

The oven has two modes - Sn60Pb40 reflow and constant temperature.
Constant temperature mode is using PID control and needs to be tuned.
